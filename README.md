# Meeting Automation System

AI-powered system to convert MS Teams meeting transcripts into Minutes of Meeting, track action items, and integrate with Jira and Confluence.

## Features

- ✅ Parse meeting transcripts
- ✅ Generate structured Minutes of Meeting
- ✅ Extract action items with assignees
- ✅ Search and reference Jira issues
- ✅ Create Confluence documentation
- ✅ MCP server integration
